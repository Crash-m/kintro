Existence Verbs (Part 2)

The English is put underneath these examples so you can see the word order explicitly.

의자에 연필 있습니다
on the chairthe pencilis
서울에 학교 있습니다
in Seoula schoolis

Note that there is no need for a subject marker on an “exists” sentence. When we use the “doesn’t exist” verb, though, we must put in a subject marker. Thus, to say, “There is no kimch’i (김치 ),” we say this:

김치가 없습니다
kimch’iis not
